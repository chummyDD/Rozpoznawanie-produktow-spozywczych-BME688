# \## Cel projektu

# Celem projektu jest zbadanie możliwości rozpoznawania produktów spożywczych czujmikiem BME688. W badaniach zostanie wykorzystana płytka uruchomieniowa "BME688 dev-kit board" z ośmioma czujnikami (https://www.bosch-sensortec.com/media/boschsensortec/downloads/product\_flyer/bst-bme688-fl001.pdf). Producent dostarcza narzędzia wspomagane sztuczną inteligencją dedykowane dla systemu uruchomieniowego. 

# 

# \## Sprzęt i Narzędzia

# W badaniach wykorzystywana jest specjalistyczna płytka uruchomieniowa wyposażona w osiem sensorów:

# \* \*\*Sprzęt:\*\* \[BME688 dev-kit board](https://www.bosch-sensortec.com/media/boschsensortec/downloads/product\_flyer/bst-bme688-fl001.pdf)

# \* \*\*Oprogramowanie:\*\* Narzędzia AI dostarczane przez producenta układu (Bosch Sensortec) dedykowane dla systemu uruchomieniowego oraz zewnętrzne frameworki do analizy danych i uczenia maszynowego.
